Experiment-10

🚀 CLI Employee Management System

This is a Node.js-based Command Line Interface (CLI) application for managing employees using in-memory arrays. The application provides an interactive terminal interface for performing CRUD operations on employee data.

🎯 Learning Objectives

  • Understanding Node.js built-in modules (readline)
  • Working with interactive CLI applications
  • Array manipulation and in-memory data storage
  • Modular code organization with require/exports
  • Error handling and input validation
  • User experience in terminal applications

🚀 How to Run

$ cd NodejsProjects/Experiment-10
$ npm start

or

$ node src/app.js

📋 Features

  • ➕ Add new employees with name and ID
  • 📋 List all employees in a formatted table
  • 🗑️ Remove employees by ID
  • 🔍 Search for employees by ID
  • 📊 Display total employee count
  • ✨ Colorful and user-friendly CLI interface
📝 Note: This is a CLI application that runs in the terminal. Open your terminal, navigate to this directory, and run the commands shown above to interact with the Employee Management System.

💡 Technologies Used

  • Node.js (Built-in modules only)
  • Readline module for user input
  • Arrays for in-memory data storage
  • ANSI color codes for terminal styling
  • Modular JavaScript with CommonJS
