🚀 CLI Employee Management System

This is a Node.js-based Command Line Interface (CLI) application for managing employees using in-memory arrays. The application provides an interactive terminal interface for performing CRUD operations on employee data.

📋 Features

  • ➕ Add new employees with name and ID
  • 📋 List all employees in a formatted table
  • 🗑️ Remove employees by ID
  • 🔍 Search for employees by ID
  • 📊 Display total employee count
  • ✨ Colorful and user-friendly CLI interface

🚀 How to Run

$ npm start

or

$ node src/app.js

💡 Technologies Used

  • Node.js (Built-in modules only)
  • Readline module for user input
  • Arrays for in-memory data storage
  • ANSI color codes for terminal styling
📝 Note: This is a CLI application that runs in the terminal. This HTML file is included for documentation purposes only. To use the application, open your terminal and run the commands shown above.

🎯 Learning Objectives

  • Understanding Node.js built-in modules
  • Working with readline for interactive CLI
  • Array manipulation and data management
  • Modular code organization
